Welcome to LetsGraphQL!

About

We all know that data is one of the most valuable commodities in the modern era and getting it to the right people at the right time and in the most appropriate format is critical for businesses.

We achieve this through Application Programming Interfaces (APIs) and there are lots of frameworks to choose from, REST, JSON-RPC, GraphQL, RabbitMQ, etc... However, it is nigh on impossible for businesses to provide each and every interface to support developers and clients in utilising their data.

Here at LetsGraphQL, we ❤ GraphQL as it communicates the underlying data model irresepctive of the backend systems that store and power it. Our mission is to make the Internet GraphQL-able. Be it through middleware on existing APIs or working directly with backend systems.

Our opensource projects are funded by our kind and generous sponsors under MIT licenses. Even our main website is opensource!

By providing GraphQL APIs to the Internets data, we aim to maximise the use of data so we can continue to see incredible applications and products that will help society.

SWAPIQL

A GraphQL wrapper around The Star Wars API (SWAPI). This is a great demonstration of how we wrap a GraphQL service around a REST API. Use the playground to interact and see what the API has to offer. The data model is visible using voyager.

The endpoint is https://api.letsgraphql.org/swapiql/graphql

Want to run the service or see the underlying code? Then head over the to docker image and GitHub repository.
