SWAPIQL

Maintainer: James Gopsill
Status: Production-Ready

A GraphQL wrapper around The Star Wars API (SWAPI). This is a great demonstration of how we wrap a GraphQL service around a REST API. Use the playground to interact and see what the API has to offer. The data model is also visible using voyager.

API endpoint: https://api.letsgraphql.org/swapiql/graphql
